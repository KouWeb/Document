# テンプレート制作のルール

-   テンプレートの役割
    -   コーダーがデザインやレイアウトを意識しなくてもページ作成ができる
-   開発環境
    -   ns-starter
-   HTML コーディング（セマンティックなマークアップの方に集約？）
    -   ほげ
-   CSS 設計
    -   基本的な考え方（予測できる・再利用できる・保守できる・拡張できる）
    -   BEM（MindBEMding）
    -   FLOCSS
-   SCSS 記述ルール
    -   コメントアウトの使い分け
    -   ネストの書き方
    -   メディアクエリの使い方
-   CSS コーディング
    -   余白の付け方
    -   ホバーリアクションを必ずつける
    -   レスポンシブ対応
        -   メディアクエリをパーツ毎に記述する
        -   フォントサイズ調整
        -   グロナビなどで PC、SP で構造が変わる場合はソースごと分ける
-   JavaScript コーディングルール
    -   厳格モード（use strict）
    -   スコープ
    -   セレクター
    -   ループ処理
-   表示速度最適化
