# モジュール設計チェックリスト

-   FLOCSS のルールに従ったファイルに定義していますか。
-   FLOCSS のルールに従った命名（プレフィックス）になっていますか。
-   BEM のルールに従った命名・構造になっていますか。
-   インタラクティブなコンテンツの場合、ホバーなどのリアクションのスタイルは設定されていますか。
-   レスポンシブ対応ができていますか。
    -   各デバイスサイズで、表示崩れは起きていませんか。
    -   フォントサイズはデザイン通りに正しく変化していますか。
    -   各余白はデザイン通りに正しく変化していますか。
    -   意図しない画像の見切れは起きていませんか。
