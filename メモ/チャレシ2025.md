## 行動目標

### 事業の視点

UI デザインサービス化に向けて、PR やオウンドメディア構築に活用できる知識・技術を身に付け、サービスとして提供できる PR 材料（サンプル）を作成する。オウンドメディアを通して外部に情報を発信・アピールし、業務獲得に繋げる。

### 人材の視点

制作業務で必要な JavaScript やアクセシビリティについて、実務レベルで扱える人員を増やす。
制作現場でよく使われる UI（例：カルーセルやタブ、アコーディオンなど）を題材に JavaScript での実装方法やアクセシブルにするための方法について勉強会を行う。
